## Pendahuluan

Gunpla, singkatan dari Gundam Plastic Model, adalah model plastik yang dikembangkan oleh Bandai Jepang. Mereka adalah replika dari robot mecha yang muncul dalam franchise Mobile Suit Gundam. Kali ini, kita akan membahas lebih lanjut tentang Gunpla, mulai dari sejarahnya, jenis-jenisnya, hingga tips dalam membangunnya.

## Sejarah Gunpla

Gunpla pertama kali diperkenalkan oleh Bandai pada tahun 1980. Awalnya, model ini hanya monokrom dan membutuhkan lem dan cat untuk merakit dan mewarnainya. Namun, seiring berjalannya waktu dan teknologi, Gunpla mulai berkembang. Pada tahun 1990, Bandai mengeluarkan seri High Grade (HG) yang lebih detail dan berwarna. Kemudian, beragam seri lainnya seperti Master Grade (MG), Perfect Grade (PG), dan Real Grade (RG) turut diluncurkan.

## Jenis-Jenis Gunpla

Ada berbagai jenis Gunpla yang berbeda, tergantung pada skala dan tingkat detailnya. Jenis yang paling populer adalah High Grade (HG), yang berukuran 1/144. HG adalah pilihan bagus untuk pemula karena cukup mudah dirakit dan harganya terjangkau. Master Grade (MG) dan Perfect Grade (PG) adalah model skala lebih besar dengan detail yang lebih baik, tetapi mereka memerlukan lebih banyak waktu dan keterampilan untuk merakit. Real Grade (RG) adalah model skala 1/144 dengan detail setara MG atau PG, membuatnya populer di kalangan builder yang lebih berpengalaman.

## Tips dalam Membangun Gunpla

Untuk membangun Gunpla, Anda perlu kesabaran dan ketelitian. Gunakan gunting model dan pisau X-Acto untuk memotong bagian dari runner. Jangan terburu-buru saat merakit, karena hal tersebut bisa merusak bagian-bagian kecil. Selain itu, gunakan pen panel lining untuk menambah detail dan kedalaman pada model Anda. Jika Anda ingin melukis model Anda, gunakan cat akrilik atau enamel dan aplikasikan dengan kuas atau airbrush.

## Kesimpulan

Gunpla adalah hobi yang menarik yang memadukan keterampilan merakit dan kreativitas. Dengan berbagai jenis dan tingkat kesulitan, ada Gunpla untuk setiap penggemar Gundam, baik pemula maupun yang berpengalaman. Jadi, selamat merakit dan nikmati prosesnya!

## Overview

* Optionally follows [GitHub Flavored Markdown](https://github.github.com/gfm/)
* Renders actual React elements instead of using `dangerouslySetInnerHTML`
* Lets you define your own components (to render `MyHeading` instead of `'h1'`)
* Has a lot of plugins

## Syntax highlighting

Here is an example of a plugin to highlight code:
[`rehype-highlight`](https://github.com/rehypejs/rehype-highlight).

```js
import React from 'react'
import ReactDOM from 'react-dom'
import Markdown from 'react-markdown'
import rehypeHighlight from 'rehype-highlight'

const markdown = `
# Your markdown here
`

ReactDOM.render(
  <Markdown rehypePlugins={[rehypeHighlight]}>{markdown}</Markdown>,
  document.querySelector('#content')
)
```

Pretty neat, eh?

## Components

You can pass components to change things:

```js
import React from 'react'
import ReactDOM from 'react-dom'
import Markdown from 'react-markdown'
import MyFancyRule from './components/my-fancy-rule.js'

const markdown = `
# Your markdown here
`

ReactDOM.render(
  <Markdown
    components={{
      // Use h2s instead of h1s
      h1: 'h2',
      // Use a component instead of hrs
      hr(props) {
        const {node, ...rest} = props
        return <MyFancyRule {...rest} />
      }
    }}
  >
    {markdown}
  </Markdown>,
  document.querySelector('#content')
)
```
